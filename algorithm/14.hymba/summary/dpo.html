학습 방법: Direct Preference Optimization (DPO)

DPO란 무엇인가?: RLHF의 우아한 대안

DPO(Direct Preference Optimization)는 인간의 선호도를 직접적으로 최적화하여 언어 모델을 정렬(Alignment)하는 혁신적인 방법론입니다. 기존의 복잡한 RLHF(Reinforcement Learning from Human Feedback) 파이프라인(보상 모델 학습 → 강화학습)을 하나의 간단한 분류 문제로 변환하여 안정성과 효율성을 크게 높였습니다.

DPO의 핵심은 보상 함수를 명시적으로 모델링하는 대신, 정책 모델과 참조 모델 간의 확률 비율을 사용하여 선호도 데이터를 직접 최적화하는 것입니다. DPO 손실 함수는 다음과 같이 표현됩니다.

$$ \mathcal{L}_{DPO}(\pi; \pi_{ref}) = - \mathbb{E}_{(x, y_w, y_l) \sim \mathcal{D}} \left[ \log \sigma \left( \beta \log \frac{\pi(y_w|x)}{\pi_{ref}(y_w|x)} - \beta \log \frac{\pi(y_l|x)}{\pi_{ref}(y_l|x)} \right) \right] $$

이 손실 함수는 모델($\pi$)이 선호되는 응답($y_w$)에 대해서는 참조 모델($\pi_{ref}$)보다 높은 확률을, 선호되지 않는 응답($y_l$)에 대해서는 낮은 확률을 부여하도록 직접적으로 학습시킵니다.

최신 연구 동향: DPO를 넘어서

DPO의 성공 이후, 그 아이디어를 확장하고 개선하려는 여러 후속 연구가 등장했습니다. 아래 차트는 각 방법론의 핵심 특징을 비교한 것입니다.

방법론 핵심 아이디어 데이터 형식
DPO 보상 모델링을 분류 문제로 변환 쌍(Chosen, Rejected)
IPO 과적합 방지를 위한 정규화 항 추가 쌍(Chosen, Rejected)
KTO '좋음/나쁨'으로 선호도 단순화 개별(Good, Bad)
